	FILEPATH
		/Android : Official Android Folder

		/atmosphere : Atmosphere Custom Firmware Folder

		/bootloader : Hekate Folder
		/bootloader/hekate_ipl.ini : .ini File for Hekate. Manage "Launch" button entries

		/config : .ini files for Atmosphere Apps (or Tesla's Overlays)
		/config/aio-switch-updater/preserve.txt : list of elements to preserve during an update.

		/emuMMC : EmuNand Folder

		/JKSV : JKSV Folder (SaveFiles Manager)

		/switch : App Folder

		/switchroot : Android: also Used by Hekate for android (Contains Android Payload and Joycons BT Dump)

		/tegraexplorer : Tegra Explorer Folder (Contains Tegra Explorer Scripts)

		/themes : Themes Folder

		/exosphere.ini : Anti-ban (Blanks ProdInfo)
		/hbmenu.nro : HomeBrew Menu app
